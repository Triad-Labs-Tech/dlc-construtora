# DLC Construtora — site

Site institucional/captação de investidores da DLC Construtora.
Projeto da **Triad Labs**.

## Estrutura
- `index.html` — a página
- `css/style.css` — todo o estilo
- `assets/diego.jpg` — foto usada na seção "Sobre"

## Rodar localmente
Abra a pasta no VS Code, instale a extensão **Live Server**,
clique com o botão direito em `index.html` → **Open with Live Server**.

## Branches
- `main` — produção (o que está no ar). Só recebe código revisado.
- `dev` — trabalho do dia a dia (branch segura).

## Publicação
Deploy automático pela Hostinger (Avançado → Git): todo merge em `main`
sobe pro ar. A branch `dev` pode ser publicada num subdomínio de staging.

Passo a passo completo no guia que acompanha o projeto.
